#  Red Hawk Wallet

Red Hawk Wallet is an Android-based digital campus payment application designed for Montclair State University students.  

The app allows students to:
- View account balance
- Generate QR ID for verification
- Perform NFC-based payments
- View transaction history
- Access digital receipt screens

---

##  Features

###  Authentication
- Login Screen
- Registration Screen
- Input validation using reusable components

### Smart Dashboard
- Displays wallet balance
- Shows recent transactions
- Empty state message when no transactions exist
- Quick access to Account (QR ID) screen

###  QR ID Screen
- Displays student ID details
- Generates account QR code
- Account selection dropdown

###  NFC Payment Flow
- NFC tap support
- Processing screen with loading animation
- Success screen with green tick
- Transaction entry creation

### Transactions
- Transaction history list
- Receipt screen UI
